# gunSelf
+ 基于gunsv3.1的改进版本 优化了缓存策略
+ 增加了工作流activiti的配置 ，
+ 支持生成字段验证表单，
+ 支持生成多行删除表格，
+ 支持自动生成各种表单控件，
+ 支持自动生成导入导出表格
+ 增加一些常用的模板文件
